### Setup .env file in _server_ folder

```bash
PORT=3000

PGUSER=...
PGPASSWORD=...
PGHOST=...
PGDATABASE=...

ARCJET_KEY=...
ARCJET_ENV=development

NODE_ENV=development
```

### Setup .env file in _client_ folder

```bash
VITE_API_URL="http://localhost:3000"
```
